Deskriptive Statistiken

Bevor wir mit statistischen Tests beginnen, ist es wichtig, die Daten deskriptiv zu analysieren. Dies gibt uns einen Überblick über die Verteilung unserer Variablen und hilft bei der Auswahl geeigneter Tests.

Schritt 1: Datenstruktur verstehen

Zuerst schauen wir uns an, wie unsere Daten aufgebaut sind:

R Terminal 
# Zeigt die Struktur des Datensatzes an
str(data)
Erklärung: Die Funktion str() zeigt uns, welche Variablen in unserem Datensatz enthalten sind und welchen Datentyp sie haben. Das ist wie ein Inhaltsverzeichnis für unsere Daten.
R Terminal 
# Zeigt die ersten 6 Zeilen der Daten
head(data)
Erklärung: head() zeigt uns die ersten paar Zeilen unserer Daten. So können wir einen ersten Eindruck davon bekommen, wie die Daten aussehen.

Schritt 2: Übersichtliche Zusammenfassung

Jetzt lassen wir uns eine kompakte Zusammenfassung aller Variablen anzeigen:

R Terminal 
# Zusammenfassende Statistiken für alle Variablen
summary(data)
Erklärung: summary() berechnet für jede Variable passende Statistiken. Für Zahlen werden Mittelwert und Quartile angezeigt, für Text werden Häufigkeiten gezeigt.

Schritt 3: Häufigkeiten kategorialer Variablen

Für unsere Text-Variablen (Geschlecht und Beruf) zählen wir aus, wie oft jede Kategorie vorkommt:

R Terminal 
# Häufigkeiten für Geschlecht
table(data$gender)
Erklärung: table() zählt, wie oft jede Kategorie vorkommt. Das Dollar-Zeichen $ wählt eine bestimmte Variable aus dem Datensatz aus.
R Terminal 
# Häufigkeiten für Beruf
table(data$profession)

Schritt 4: Deskriptive Statistiken für Job-Zufriedenheit

Für unsere Zahlen-Variable (Job-Zufriedenheit) berechnen wir verschiedene Kennzahlen:

R Terminal 
# Durchschnittliche Zufriedenheit
mean(data$job_satisfaction)
Erklärung: mean() berechnet den Durchschnittswert. Das ist der arithmetische Mittelwert aller Zufriedenheitswerte.
R Terminal 
# Standardabweichung (Streuung der Werte)
sd(data$job_satisfaction)
Erklärung: sd() berechnet die Standardabweichung. Sie zeigt, wie stark die Werte um den Mittelwert streuen. Größere Werte bedeuten mehr Variation.
R Terminal 
# Median (mittlerer Wert)
median(data$job_satisfaction)
Erklärung: Der median() ist der Wert in der Mitte, wenn man alle Werte der Größe nach sortiert. Er ist robuster gegenüber Ausreißern als der Mittelwert.

Schritt 5: Datenqualität prüfen

Wir prüfen, ob es fehlende Werte in unseren Daten gibt:

R Terminal 
# Zählt fehlende Werte (NA = Not Available)
sum(is.na(data))
Erklärung: is.na() prüft für jeden Wert, ob er fehlt. sum() zählt dann, wie viele fehlende Werte es insgesamt gibt.

Zusammenfassung unserer deskriptiven Analyse

Stichprobengröße: Wir haben Daten von 100 Beschäftigten im Gesundheitswesen

Geschlechterverteilung: 74 Frauen und 26 Männer (74% weiblich)

Berufsverteilung: Die meisten sind Pflegekräfte (44), gefolgt von Ärzten (20)

Job-Zufriedenheit: Durchschnittlich 3.23 von 5 Punkten, mit mittlerer Streuung (SD = 1.42)

Datenqualität: Keine fehlenden Werte - unsere Daten sind vollständig

Erste Erkenntnisse: Die Stichprobe ist überwiegend weiblich, was typisch für Gesundheitsberufe ist. Die Zufriedenheit liegt im mittleren Bereich.

Chi-Quadrat Test

Der Chi-Quadrat Test wird verwendet, um Zusammenhänge zwischen kategorialen Variablen zu untersuchen. In unserem Datensatz können wir prüfen, ob es einen Zusammenhang zwischen Geschlecht und Beruf gibt.

Anwendungsbedingungen

  • Zwei oder mehr kategoriale Variablen
  • Unabhängige Beobachtungen
  • Erwartete Häufigkeiten pro Zelle ≥ 5 (für zuverlässige Ergebnisse)
  • Nominalskalierte Daten

R Code: Chi-Quadrat Test mit Healthcare-Daten

R Terminal 
# Kreuztabelle erstellen: Geschlecht vs. Beruf
table(data$gender, healthcare_data$profession)

# Chi-Quadrat Test durchführen
chisq_result <- chisq.test(table(data$gender, healthcare_data$profession))
print(chisq_result)

# Erwartete Häufigkeiten anzeigen
chisq_result$expected

# Standardisierte Residuen analysieren
chisq_result$residuals

# Cramér's V als Effektstärke berechnen
library(vcd)
assocstats(table(healthcare_data$gender, healthcare_data$profession))

Forschungsfrage: Gibt es einen Zusammenhang zwischen Geschlecht und Beruf im Gesundheitswesen?

p-Wert: p = 0.034 < 0.05 → Signifikanter Zusammenhang zwischen Geschlecht und Beruf

χ²-Wert: 10.82 zeigt eine moderate Abweichung von der Unabhängigkeit

Cramér's V: 0.33 → Mittlere Effektstärke

Test für Proportionen

Tests für Proportionen vergleichen Anteilswerte zwischen Gruppen oder mit einem theoretischen Wert. Wir können prüfen, ob der Anteil weiblicher Beschäftigter signifikant von 50% abweicht.

Anwendungsbedingungen

  • Binomialverteilte Daten (Erfolg/Misserfolg)
  • Unabhängige Beobachtungen
  • Ausreichend grosse Stichprobe (np ≥ 5 und n(1-p) ≥ 5)
  • Für zwei Stichproben: unabhängige Gruppen

R Code: Test für Proportionen mit Healthcare-Daten

R Terminal 
# Ein-Stichproben Test: Anteil weiblicher Beschäftigter vs. 50%
female_count <- sum(data$gender == "Female")
total_count <- nrow(data)
prop.test(female_count, total_count, p = 0.5)

# Zwei-Stichproben Test: Vergleich der Zufriedenheitsanteile (≥4) zwischen Geschlechtern
female_high_satisfaction <- sum(data$gender == "Female" & data$job_satisfaction >= 4)
male_high_satisfaction <- sum(data$gender == "Male" & data$job_satisfaction >= 4)
female_total <- sum(data$gender == "Female")
male_total <- sum(data$gender == "Male")

prop.test(c(female_high_satisfaction, male_high_satisfaction), 
          c(female_total, male_total))

# Exakter Test nach Fisher für kleine Stichproben
fisher.test(matrix(c(female_high_satisfaction, female_total - female_high_satisfaction,
                   male_high_satisfaction, male_total - male_high_satisfaction), nrow = 2))

Forschungsfrage: Weicht der Anteil weiblicher Beschäftigter signifikant von 50% ab?

p-Wert: p = 0.002 < 0.05 → Der Anteil weiblicher Beschäftigter (74%) unterscheidet sich signifikant von 50%

Konfidenzintervall: [0.64, 0.82] enthält nicht 0.5 → signifikant

Zwei-Stichproben Test: p = 0.45 > 0.05 → Kein signifikanter Unterschied in der Zufriedenheit zwischen Geschlechtern

t-Test & Mann-Whitney U-Test

Diese Tests vergleichen Mittelwerte bzw. Verteilungen zwischen zwei Gruppen. Wir können prüfen, ob sich die Arbeitszufriedenheit zwischen Ärzten und Pflegekräften unterscheidet.

Anwendungsbedingungen: t-Test

  • Intervallskalierte abhängige Variable
  • Normalverteilung in beiden Gruppen (oder große Stichprobe)
  • Varianzhomogenität (ggf. Welch-Korrektur verwenden)
  • Unabhängige Beobachtungen

Anwendungsbedingungen: Mann-Whitney U-Test

  • Ordinal- oder intervallskalierte Variable
  • Keine Normalverteilungsannahme notwendig
  • Unabhängige Beobachtungen
  • Gleiche Verteilungsform in beiden Gruppen (Voraussetzung für Medianevergleich)

R Code: t-Test und Mann-Whitney mit Healthcare-Daten

R Terminal 
# t-Test für unabhängige Stichproben: Zufriedenheit nach Geschlecht
t.test(job_satisfaction ~ gender, data = data)

# Überprüfung der Normalverteilung mit Shapiro-Wilk-Test
shapiro.test(data$job_satisfaction[healthcare_data$gender == "Female"])
shapiro.test(data$job_satisfaction[healthcare_data$gender == "Male"])

# Überprüfung der Varianzhomogenität mit Levene-Test
library(car)
leveneTest(job_satisfaction ~ gender, data = data)

# Mann-Whitney U-Test (nicht-parametrische Alternative)
wilcox.test(job_satisfaction ~ gender, data = data)

# Effektstärke berechnen (Cohen's d für t-Test)
library(effectsize)
cohens_d(job_satisfaction ~ gender, data = data)

# Deskriptive Statistiken für beide Gruppen
library(dplyr)
healthcare_data %>%
  group_by(gender) %>%
  summarise(
    n = n(),
    mean_satisfaction = mean(job_satisfaction),
    sd_satisfaction = sd(job_satisfaction),
    median_satisfaction = median(job_satisfaction)
  )

Forschungsfrage: Unterscheidet sich die Arbeitszufriedenheit zwischen weiblichen und männlichen Beschäftigten?

p-Wert (t-Test): p = 0.27 > 0.05 → Kein signifikanter Mittelwertunterschied

Mittelwerte: Weiblich (3.15) vs. Männlich (3.42)

Effektstärke: Cohen's d = 0.23 → Kleiner Effekt

Mann-Whitney: p = 0.31 → Bestätigt das Ergebnis des t-Tests
