R-Skripte

Ein R-Skript speichert Ihren Code in einer Datei (.R). Sie können ihn später erneut ausführen oder teilen. Tipp 


# Speichern Sie diesen Code als "mein_skript.R"
library(ggplot2)
daten <- read.csv("meine_daten.csv")
ggplot(daten, aes(x = Variable1, y = Variable2)) +
  geom_point()
