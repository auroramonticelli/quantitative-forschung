Daten prüfen

Bevor wir visualisieren, überprüfen wir die Daten auf fehlende Werte und typische Fehler.Tipp


# Daten anschauen
head(mtcars)
summary(mtcars)

Warum visualisieren?

Visualisierung hilft, Muster, Ausreißer und Zusammenhänge zu erkennen.Tipp


# Beispiel: Scatterplot erkennen Muster
plot(mtcars$hp, mtcars$mpg)

ggplot2 & Cheatsheet

Wir nutzen ggplot2 für ansprechende Grafiken. Cheatsheets helfen, schnell Funktionen zu finden.Tipp


library(ggplot2)

Plots: Barplot, Boxplot, Histogram

Wir erstellen verschiedene Plottypen, um Daten zu erkunden.Tipp


# Barplot
ggplot(mtcars, aes(x = factor(cyl))) + geom_bar(fill = "#b9aadf")

# Boxplot
ggplot(mtcars, aes(x = factor(cyl), y = mpg)) + geom_boxplot(fill = "#b9aadf")

# Histogram
ggplot(mtcars, aes(x = mpg)) + geom_histogram(fill = "#b9aadf", bins = 10)

Ästhetische Anpassungen

Wir passen Titel, Farben und Positionen an, um Grafiken klarer zu gestalten.Tipp


ggplot(mtcars, aes(x = hp, y = mpg)) +
  geom_point(color = "#b9aadf") +
  geom_smooth(method = "lm", se = FALSE, color = "#9b7bb5") +
  labs(
    title = "Horsepower vs. MPG",
    subtitle = "Beispiel mit angepasster Schriftgröße",
    x = "Horsepower (HP)",
    y = "Miles per Gallon (MPG)"
  ) +
  theme(
    plot.title = element_text(size = 20, face = "bold", color = "#9b7bb5"),
    plot.subtitle = element_text(size = 16, face = "italic", color = "#b9aadf")
  )
