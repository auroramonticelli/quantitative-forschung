Overview

We use ggplot2 in R to create professional and insightful graphics. This section covers scatterplots, boxplots, and histograms.

Example: Scatterplot


library(ggplot2)

ggplot(mtcars, aes(x = hp, y = mpg)) +
  geom_point(color = "#0ea5e9") +
  geom_smooth(method = "lm", se = FALSE, color = "#1e3a8a") +
  labs(title = "Horsepower vs. MPG")
