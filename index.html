Course Description

This session introduces students to the fundamentals of quantitative research using R. We will focus on two key aspects: creating clear and informative visualizations, and conducting statistical tests to answer research questions.

Visualization

Students will learn the basics of data visualization in R. We will use ggplot2 to create plots such as:

  • Histograms and bar charts
  • Boxplots and scatterplots
  • Adding titles, labels, and themes

Example in R:


library(ggplot2)

# Example: Scatterplot with regression line
ggplot(mtcars, aes(x = hp, y = mpg)) +
  geom_point() +
  geom_smooth(method = "lm", col = "blue") +
  labs(title = "Horsepower vs. MPG")

Testing

We will introduce basic statistical tests in R, focusing on how to apply them and interpret the results:

  • t.test() for comparing means
  • cor.test() for correlations
  • chisq.test() for categorical data

Example in R:


# Example: t-test
t.test(mpg ~ am, data = mtcars)
