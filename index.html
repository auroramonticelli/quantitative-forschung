Course Overview

This one-session crash course introduces the essentials of quantitative research with R. You will learn how to create effective visualizations and how to apply basic statistical tests to answer research questions.

Visualization

Data becomes meaningful when it is well presented. We will use ggplot2 to create clear, professional visualizations such as:

  • Scatterplots
  • Boxplots
  • Histograms

library(ggplot2)

ggplot(mtcars, aes(x = hp, y = mpg)) +
  geom_point(color = "#ff5c8d") +
  geom_smooth(method = "lm", se = FALSE, color = "#6a11cb") +
  labs(title = "Horsepower vs. MPG")

Statistical Testing

To complement visualization, we will cover basic statistical tests in R:

  • t.test() — Compare means
  • cor.test() — Assess correlations
  • chisq.test() — Work with categorical data

# Example: t-test
t.test(mpg ~ am, data = mtcars)
