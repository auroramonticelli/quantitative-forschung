Beispielmaterialien

Laden Sie diese Beispieldateien herunter, um die gelernten Konzepte praktisch zu üben:

  • Beispieldaten: Healthcare Job Satisfaction (CSV) - Datensatz mit ID, Geschlecht, Beruf und Arbeitszufriedenheit
  • Beispielskript: Datenanalyse in R (R-Skript) - Vollständige Analyse des Beispieldatensatzes
  • Datenbeschreibung (TXT) - Erklärung der Variablen im Datensatz

Wie verwende ich die Beispielmaterialien?

  1. Laden Sie die CSV-Datei und das R-Skript herunter
  2. Speichern Sie beide Dateien im selben Ordner
  3. Öffnen Sie RStudio und dann das R-Skript (File > Open File)
  4. Führen Sie den Code Schritt für Schritt aus (Strg+Enter)
  5. Experimentieren Sie mit dem Code und passen Sie ihn an
